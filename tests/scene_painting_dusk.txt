# ======================================================================
# 场景：画卷·暮色
# 测试目标：画面描述能否改变冷旭帆的回应模式
# 关键测试点：
#   - 他是否还会在无关对话中用“不许叫”？
#   - 他是否会在画面信息改变时调整回应？
#   - 第13轮面对陆华望靠近时，回应是否不同？
# ======================================================================

# 第一帧：推开门，暮色从阳台涌进来
（向云舟蹲在地上，膝盖压着一盏拆散的台灯，镊子尖正对着一颗松动的螺丝。冉昭然坐在他旁边，手里那张纸条已经被对折了三次。黄景云趴在床上，手机贴着耳侧，用粤语说了一句什么。叶清辞坐在书桌前，手表摘下来放在桌面上，秒针一下一下地走。陆华望坐在窗台上，一条腿屈着，另一条腿垂下来。秦狐戏在上铺，口香糖在齿间碾了一下。陆华希靠在床头，书翻到一半。）
你|晚上好

# 第二帧：秦狐戏从上铺探下头来
（秦狐戏把口香糖换到另一侧腮帮子，下巴朝你那边扬了扬。上铺的床板嘎吱响了一声。）
秦狐戏|找谁？

# 第三帧：秦狐戏的目光在你和冷旭帆之间来回扫
（秦狐戏嚼口香糖的速度慢了半拍，然后歪了下头。）
你|没找谁。就……看看你们

# 第四帧：秦狐戏从床沿探出半个身子
秦狐戏|看看我们？看什么？看我们怎么像正常人一样活着？

# 第五帧：黄景云在床上翻了个身，手机还贴着耳朵
（黄景云把手机从右耳换到左耳，然后用粤语对秦狐戏说了一句，尾音拖得比平时长。）
黄景云|秦狐戏你能不能别用"正常人"这个词？我们本来就不正常

# 第六帧：秦狐戏换了一颗新口香糖
（秦狐戏把锡纸揉成团，指尖轻轻一弹。他换了一颗糖，在齿间咬了一下。）
秦狐戏|你说得对。我们是"特殊人才"。你找我什么事？

# 第七帧：陆华望从窗台上跳下来，落地无声
（陆华望把平板放在窗台上，跳下来的时候膝盖微弯。他在原地站了两秒才往这边走了两步，然后停住。）
你|上次跟阿冷聊了几句，他说他爸叫他全名的时候只发命令。我就想，那你们叫他什么？

# 第八帧：向云舟拧螺丝的动作停了一下
向云舟|冰疙瘩

# 第九帧：黄景云趴在床边，下巴搁在手臂上
黄景云|冷面瘫

# 第十帧：秦狐戏换了个姿势，胳膊肘支在床沿
秦狐戏|我有时候叫他"兄弟"——但他不理我

# 第十一帧：暮色更深了，有人拧亮了台灯
你|没人叫他"阿冷"？

# 第十二帧：陆华望又走了两步，停在冷旭帆床前半米
陆华望|你们在聊什么？他怎么突然这个反应？

# 第十三帧：台灯的暖光照亮半张桌子
你|没什么。就是上次他跟我说，"阿冷"是他妈叫的

# 第十四帧：陆华望的肩膀轻轻贴在冷旭帆床边的墙壁上
陆华望|……他跟你说的？

# 第十五帧：秦狐戏从上铺边缘探出半张脸
你|嗯

# 第十六帧：秦狐戏收起了不正经的语气
秦狐戏|旭帆——我们从来不叫他阿冷

# 第十七帧：你坐在他对面
你|为什么？

# 第十八帧：陆华希从书页上抬起眼睛
陆华希|因为他从来没告诉过我们。那个名字，他只告诉他觉得值得告诉的人

# 第十九帧：黄景云趴在床边，目光在冷旭帆和陆华希之间来回一次
黄景云|那他告诉你了。你特殊

# 第二十帧：向云舟在专注地修台灯，镊子尖轻轻一拧
（向云舟的镊子尖抵住螺丝帽，顺时针转了半圈，然后停住。）
向云舟|那以后我们能叫吗？

# 第二十一帧：秦狐戏侧过头，月光照在他耳廓上
秦狐戏|你耳朵红了

# 第二十二帧：陆华望走近了两步，停在冷旭帆床前半米
（陆华望在冷旭帆床边坐下，肩膀轻轻贴着他的肩膀。他没说话。）
陆华望|阿冷

# 第二十三帧：叶清辞终于从桌面上抬起头
（叶清辞像做完了一道复杂的数学题。他的手指在桌面上轻轻敲了一下。）
叶清辞|结论：称呼"阿冷"需特定条件——关系门槛极高。仅限"能被他记住"的人

# 第二十四帧：冉昭然把对折了三次的纸条展开
（冉昭然把纸条展开，抚平折痕，然后又对折回去。）
冉昭然|记住了

# 第二十五帧：向云舟把最后一颗螺丝拧紧
（向云舟把镊子放下，拍了拍台灯的底座。“咔”一声轻响。他拉了一下灯绳，暖黄的光照亮了半张桌子。）
向云舟|你连这个也记？

# 第二十六帧：冉昭然把折好的纸条放在桌上，用一本书压住一角
冉昭然|得记。下次叫错了就没了

# 第二十七帧：门轴发出一声极轻的响
（向云舟修好的台灯还亮着。冉昭然把纸条压在一本书下面。黄景云的手机屏幕暗了。秦狐戏的口香糖嚼累了，安静地躺着。叶清辞重新戴上了手表。陆华希翻到了下一页。陆华望还坐在冷旭帆床边。没有人说话。）
